Computer vision based hand gesture recognition for interacting with 3D artwork

April 2026 - Ongoing

This project is focused on developing a computer vision based hand gesture recognition system for interacting with 3D artwork in a virtual environment. The goal is to create an system that allows users to manipulate 3D art using their hand movements.

This work is currenly in progress. An update will follow after its completion, which is expected to be around the end of 2026.
